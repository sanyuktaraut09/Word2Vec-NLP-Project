## Overview

This project demonstrates an end-to-end Natural Language Processing (NLP) workflow using stock headline data. It focuses on text preprocessing, word embedding generation using Word2Vec, and comparison of CBOW and Skip-Gram models for learning semantic word representations.

---

## Features

- Text preprocessing using NLTK
- Stopword removal
- Lowercase conversion
- Punctuation removal
- Tokenization
- Word2Vec implementation
- CBOW model
- Skip-Gram model
- Embedding visualization
- Semantic comparison

---

- Python
- NLTK

- Scikit-learn

## Dataset

Stock headline sentiment dataset containing news headlines and sentiment labels.

---

## Workflow

1. Data preprocessing
2. Tokenization
3. Stopword removal
4. CBOW training
5. Skip-Gram training
6. Embedding visualization
7. Semantic analysis

---

## Models

### Continuous Bag of Words (CBOW)

Predicts the target word using surrounding context.

### Skip-Gram

Predicts surrounding context using the target word.

---

## Results

- Generated semantic word embeddings
- Compared CBOW and Skip-Gram representations
- Visualized embedding vectors
- Analyzed semantic similarities
